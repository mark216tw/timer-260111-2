# Bauhaus 倒數計時器

一個致敬包浩斯（Bauhaus）風格的倒數計時器，強調「形隨機能」的幾何美學。結合了強烈的原色對比、幾何圖形與現代 Web 技術，提供美觀且高效率的計時體驗。

## ✨ 主要特色

### 🎨 包浩斯設計美學 (Bauhaus Design)
- **幾何構成**：純粹的圓形、方形與三角形構成畫面。
- **原色對比**：經典的紅 (#D02020)、藍 (#1040C0)、黃 (#F0C020) 三原色搭配極致黑白。
- **實用主義**：版面緊湊，鎖定 100vh 視窗高度，資訊一目瞭然，無需捲動。
- **高對比模式**：
  - **白晝**：米白底色，黑色粗框，強烈視覺衝擊。
  - **黑夜**：深黑底色，白色線條，保留原色點綴，舒適護眼。

### 🎯 核心功能
- **彈性時間設定**：時、分、秒獨立調整
- **快速預設**：一鍵設定 5/10/15/25/30 分鐘
- **幾何進度環**：幾何線條構成的倒數進度指示
- **音效提示**：操作回饋音效 + 計時結束鬧鈴 (Web Audio API)
- **自訂標題**：可編輯計時器名稱，風格化的輸入介面

### 🔗 URL 狀態共享
- **跨裝置同步**：複製 URL 即可在任何裝置繼續倒數
- **即時狀態追蹤**：URL 自動包含計時狀態（進行中/暫停）
- **精確時間計算**：基於時間戳記，確保跨裝置時間一致
- **標題同步**：自訂標題也會包含在分享連結中

**URL 參數說明：**
- `t` - 初始時間（秒數）
- `s` - 狀態（`running` 或 `paused`）
- `start` - 開始時間戳記
- `title` - 計時器標題

### ⌨️ 鍵盤快捷鍵
- `Space` - 開始/暫停計時
- `↑` - 增加 1 分鐘
- `↓` - 減少 1 分鐘
- `R` - 重設計時器

## 📖 操作說明

1. **設定時間**：點擊幾何方塊按鈕 `+` / `-` 或使用預設時間標籤。
2. **開始/暫停**：按下巨大的「START」/「PAUSE」色塊按鈕。
3. **重設**：按下紅色的「RESET」按鈕。
4. **主題切換**：點擊右上角的幾何圖示切換日夜模式。

## 🛠️ 技術架構

- **純前端實現**：單一 HTML 檔案，無需後端。
- **零依賴**：不使用任何框架 (No React, No Vue, No Tailwind)，純原生 HTML/CSS/JS。
- **RWD 響應式**：
  - **桌面版**：粗框線 (4px)，大氣的留白。
  - **行動版**：細框線 (2px)，緊湊佈局，適應單手操作。
- **字體**：使用 Google Fonts **'Outfit'** (幾何無襯線體) 搭配 **'Noto Sans TC'**。

## 📦 安裝與使用

1. 下載 `index.html`。
2. 直接用瀏覽器開啟即可。
3. 或部署至 GitHub Pages / Vercel / Netlify 分享給朋友。
