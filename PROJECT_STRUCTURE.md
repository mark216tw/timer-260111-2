## 檔案說明

### 核心檔案

#### `index.html` (23KB)
**用途**：完整的倒數計時器應用程式

**內容結構**：
- **HTML**：頁面結構（~70 行）
  - Header：標題 + 主題切換按鈕
  - Timer Container：SVG 進度環 + 時間顯示
  - Controls：時間調整器 + 預設按鈕
  - Actions：開始/暫停、重設按鈕

- **CSS**：樣式定義（~400 行）
  - CSS 變數（主題系統）
  - Glassmorphism 效果
  - 響應式設計
  - 動畫定義

- **JavaScript**：應用邏輯（~280 行）
  - 計時器核心邏輯
  - URL 狀態管理
  - 主題切換
  - 音效系統
  - LocalStorage 整合
  - 鍵盤快捷鍵

**特點**：
- ✅ 零外部依賴
- ✅ 可直接在瀏覽器開啟
- ✅ 單檔案易於分享和部署
